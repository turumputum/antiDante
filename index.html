⚠ GStreamer не найден

Для работы приложения необходим GStreamer с поддержкой WASAPI2.

Инструкция по установке:

  1. Скачайте GStreamer (MSVC 64-bit) с официального сайта:
    https://gstreamer.freedesktop.org/download/
  2. Установите оба пакета: runtime и development
  3. При установке выберите Complete (полная установка)
  4. Добавьте C:\gstreamer\1.0\msvc_x86_64\bin в системную переменную PATH
  5. Перезапустите приложение
